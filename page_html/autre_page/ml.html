Installation

Le chatbot à besoin de modules en plus qui ne sont pas intégrées dans python, ces modules sont essentiellement destinées à simplifier le programme.Un programme nommé setup.bat permet d'installer tout les modules nécessaires en seulement 2 cliques. Si vous voulez faire les intallations manuellement il faudra donc installer :


- nltk avec pip install nltk

- numpy avec pip install numpy

- os avec pip install os

- tkinter avec pip install tkinter

- json avec pip install json

- tflearn avec pip install tflearn

- tensorflow avec pip install tensorflow


si vous disposé d'un GPU nvidia qui contient des cudas cores vous pouver innstaller tensorflow-gpu avec pip isntall tensorflow-gpu cela permeterra d'utilisé la puissance de calcule de votre gpu pour entrainée le chatbot.



Nous vous recommandons d'utiliser google colab et d'activé l'accelration par GPU ou TPU

Démarche

Le principe du chat bot est de calculer la probabilité de toute les réponses disponibles et de choisir celle qui est la plus probable d'entre elle selon lui.


Pour cela on analyse tout les mots qui sont dans la phrase donné pour pouvoir ensuite regarder combien de mots sont en commun dans la phrase donné et dans les phrases du data qui sert à entrainée l'IA. L'IA va permettre de comprendre les racines des mots de savoir que gentillement peu vouloir dire gentillement et de devinée qu'elles sont les mots important de la phrase, il va donc éliminé par lui même les : un, une, de, les, le, la... . L'intérait d'utilisé l'IA est qu'elle va d'elle même deviné les règles, ce qui va lui permettre d'être flexibles sur les questions posée et donc de donnée une réponses aproprié.

Le fonctionnement de l'IA

Pour ce chat bot le model choisie d'IA est le supervised learning.


Le supervised learning est une technique d'entrainement d'IA qui conssiste à donner les entrés et les sorties à un programme qui va posé les questions à l'IA qui va donné une réponse, lorsque la réponse est fausse l'IA enregistrera que cette réponse est fausse et qu'il faut donc donné une autre réponse. Lorsqu'elle a juste elle stocke la réponse juste pourvoir la redonné au moment où on lui demande.
