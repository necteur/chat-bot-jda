Installation

Le chatbot a besoin de modules en plus qui ne sont pas intégrées dans python, ces modules sont essentiellement destinés à simplifier le programme. Un programme nommé setup.bat ou setup.py permet d'installer tous les modules nécessaires en seulement 2 clics. Si vous voulez faire les intallations manuellement il faudra donc installer :


- nltk avec pip install nltk

- numpy avec pip install numpy

- os avec pip install os

- tkinter avec pip install tkinter

- json avec pip install json

- tflearn avec pip install tflearn

- tensorflow avec pip install tensorflow


si vous disposez d'un GPU nvidia qui contient des CUDA cores vous pouver installer tensorflow-gpu avec pip isntall tensorflow-gpu cela permettra d'utiliser la puissance de calcul de votre carte graphique (GPU) pour entrainer le chatbot.



Nous vous recommandons d'utiliser google colab et d'activer l'accellération par GPU ou TPU


Retrouver toutes les informations ici dans le README.txt

Google Colab

Nous vous conseillons d'utiliser google colabotory pour tester l'entrainement pour 2 raisons :

  • la rapidité d'entrainement
  • la précision de l'entrainement qui est bien meilleur grace à l'utilisation de puissant ASIC spécialisé pour tensorflow (TPU)

Mais cela a un inconvéniant, Google colabotory ne prend pas en charge les interfaces graphique, c'est pour cela que nous vous le recommandons que si vous voulez entrainer l'IA un grand nombre de fois

Cliquez ici pour accéder au notebook Google colabotory (il faudra se connecter avec un compte google pour pouvoir éxécuter le code). Penser à vérifier que l'accélration TPU est bien activé (Modifiée ==> paramètre du notebook ==> accélration materiel ==> TPU) et que le train.json est bien importer (il suffit de le glissé dans la bare de gauche lorsque la catégorie fichier est sélectionner).

Démarche

Le principe du chat bot est de calculer la probabilité de toute les réponses disponibles et de choisir celle qui est la plus probable d'entre elle selon lui.


Stockage des données d'entrainement

Nous avons donc commencer par créer un fichier json qui contient toute les donnés nécessaires pour l'entrainement du chat bot. Nous avons donc mis dans le fichier train.json un dictionnaire python qui contient : un tag, les questions types et leurs réponses.

Traitement des données

Pour analysé les données, on analyse tout les mots qui sont dans la phrase donné pour pouvoir ensuite regarder combien de mots sont en commun dans la phrase donnée et dans les phrases du data qui servent à entrainer l'IA. L'IA va permettre de comprendre les racines des mots de savoir que gentillement peu vouloir dire gentillement et de devinée qu'elles sont les mots important de la phrase, il va donc éliminé par lui même les : un, une, de, les, le, la... . L'intérait d'utilisé l'IA est qu'elle va d'elle même deviné les règles, ce qui va lui permettre d'être flexibles sur les questions posée et donc de donnée une réponses aproprié.

Le fonctionnement de l'IA

Pour ce chat bot le model choisie d'IA est le supervised learning.


Le supervised learning est une technique d'entrainement d'IA qui conssiste à donner les entrés et les sorties à un programme qui va posé les questions à l'IA qui va donné une réponse, lorsque la réponse est fausse l'IA enregistrera que cette réponse est fausse et qu'il faut donc donné une autre réponse. Lorsqu'elle a juste elle stocke la réponse juste pourvoir la redonné au moment où on lui demande.
